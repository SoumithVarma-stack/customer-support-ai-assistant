# Assumptions

These are the main assumptions for this proof of concept.


## Local Storage

Local JSON files are enough for this proof of concept.

This includes:

- support data
- audit logs

## Scope

The app is focused on simple e-commerce support cases like:

- orders
- shipping
- tracking
- refunds
- cancellations
- billing
- account access

## Development Setup

The app is expected to run locally with two servers:

- backend on port `5001`
- frontend through Vite
