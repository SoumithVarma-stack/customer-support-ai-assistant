# Future Improvements

These are useful next steps if this proof of concept becomes a larger app.

## Better Search

- Add semantic search with embeddings.
- Use a vector database.
- Rank results by meaning, not only keyword matches.
- Improve handling for synonyms and vague wording.

## Better Knowledge Base

- Add a larger e-commerce support knowledge base.
- Use only approved support articles and policies.
- Add versioning for support content.
- Show which exact article was used for each draft.

## Better Storage

- Move audit logs from local JSON to a database.
- Store conversations and decisions in a safer way.
- Add backups and retention rules.

## Feedback Analytics

- Use feedback to improve prompts and data.
