# Assignment Requirements

Build a Customer Support AI Assistant proof of concept.

Core requirements:
- React + TypeScript frontend.
- Node.js + Express backend.
- Agent can enter a customer query.
- Backend searches local support data like FAQs, policies, and sample tickets.
- Backend calls an LLM to generate a draft response.
- The AI response is only a suggestion, not auto-sent.
- Agent can edit the draft.
- Agent can approve or reject the draft.
- Approved or rejected decisions must be saved in a local backend audit log file.
- Include feedback handling.
- Include conversation/history/status indicators.
- Use only public, synthetic, or de-identified data.
- App must run locally with documented commands.
- Include README with setup, run commands, LLM API key setup, dataset details, architecture, assumptions, limitations, responsible AI notes, and future improvements.
- Maintain meaningful Git commit history.
